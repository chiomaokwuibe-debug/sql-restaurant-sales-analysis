# sql-restaurant-sales-analysis
SQL project analyzing restaurant sales data using SELECT queries to extract product revenue insights and support business decision-making.
🍽️ Restaurant Sales Analysis (SQL)

This project demonstrates the use of SQL queries to analyze restaurant sales data.

Skills Demonstrated:
• SELECT statements
• Data retrieval
• Revenue analysis
• Database querying
• Basic business insights

Tools Used:
• MySQL
• SQL

Sample Query:
SELECT PRODUCT, REVENUE
FROM practice_db.restaurant;

Insights:
• Identified revenue generated by different food products.
• Compared sales performance across menu items.
• Extracted data for business reporting and analysis.
